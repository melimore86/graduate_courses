Create AIC table

Description

Create an AIC table of results from a list of previously run models

Usage

createAicTable(mod.list, use.aicc = FALSE)

Arguments

mod.list

a list of objects of class occMod, i.e., the output from the function occMod.

use.aicc

logical argument for whether small-sample adjustment should be used with AIC. Effective sample set to number of units. ... OR... user-defined value for effective sample size. So, possibilities are:

  • 0 or FALSE: Use AIC

  • 1 or TRUE : Use number of sites for eff. sample size to calculate AICc

  • n : Use n for eff. sample size to calculate AICc

Value

Returns a list with the following components:

  • table - A data frame that includes:

    • the name

    • AIC value

    • relative difference in AIC

    • AIC model weight

    • -2*log-likelihood

    • number of parameters

    • any warnings for each model

    Table entries are ranked according to AIC.

  • models - List of models supplied in mod.list, ordered by AIC

  • ess - Effective sample size for AICc calculation (0=use AIC, >0=eff. samp. size)
