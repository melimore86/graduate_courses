psi_cond {RPresence}R Documentation

get conditional occupancy estimates from single-season ("so") model

Description

psicond Computes conditional (on detection history) occupancy estimates from single-season ("so") model. Uses "delta" method, with numerical gradient.

Usage

psi_cond(m0, paodata)

Arguments

m0

model object (result from running "occMod").

paodata

pao data object

Value

numeric matrix with column of estimates and column of std. errors
